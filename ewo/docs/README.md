# Docs（工程ナビ）

このディレクトリには  
EWO Factory の **作業工程ナビゲーション**を格納します。

docs は **spec の代替ではありません。**

spec に書かれている内容を  
作業工程ごとに **参照しやすく整理したもの**です。

---

# 工程

EWO作業は次の工程で進みます。

1. PLAN
2. DO
3. CHECK
4. ACTION

---

# 各工程の参照先

## PLAN

EWO設計を行う工程です。

参照

- pdca/PLAN.md

---

## DO

EWO生成を行う工程です。

参照

- pdca/DO.md

---

## CHECK

生成結果の確認を行います。

参照

- pdca/CHECK.md
---

## ACTION

問題があった場合の改善工程です。

参照

- pdca/ACTION.md

---

# 役割

docs は

- 作業手順の入口
- spec の参照ナビ
- 工程ごとの参照整理

として利用します。
