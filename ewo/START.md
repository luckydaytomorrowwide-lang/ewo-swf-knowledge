# EWO Knowledge Entry Point

このリポジトリは EWO Factory の知識ベースです。

すべての処理は次の順序で参照してください。

1. spec
   仕様定義

2. decisions
   設計判断

3. examples
   実装例

4. templates
   生成テンプレート

5. checklists
   検証

---

## Mandatory Rule

AIは必ず次の順序で参照すること

spec → decisions → examples → templates → checklists

specが最優先である。

---

## Purpose

SWF生成を **再現可能なプロセス**として固定する。
